## 注释

### 单行注释

#### lua
 
 > 跟sql注释方式类似
 
 ```lua
 -- 单行注释
print("hello lua")
 ```

 #### go java
 
 > go和java的注释一样，下面 

 ```go
 // 单行注释
 fmt.Println("hello go")
 ```

 ### 多行注释

 #### lua

```lua
 --[[
     多行注释
 --]]
 print("hello lua")
 ```

#### java go

```go
/*
 被/**/包围
*/
fmt.Println("hello go")
```
